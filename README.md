# all-3-of-1.2

learning outcomes

Understanding Frontend Development
Learned how to design interactive user interfaces using HTML, CSS, and JavaScript for projects like the live counter, SVG drawing tool, and product filter.
Improved DOM Manipulation Skills
Gained experience in dynamically updating webpage content using JavaScript, such as increasing/decreasing counter values, filtering products, and drawing SVG shapes.
Event Handling and User Interaction
Developed skills in handling user actions like button clicks, mouse movements, input selection, and real-time updates for better user experience.
Working with SVG Graphics
Understood how SVG elements work for creating and manipulating shapes, lines, and drawings directly in the browser.
Data Filtering and Logic Building
Learned how to apply filtering logic to display products based on categories, price, or search input efficiently.
Backend Integration Basics
Understood how frontend and backend communicate in full stack development, including handling data storage and retrieval where required.
